El usuario debe utilizar el Sitio Web conforme a la ley, las buenas costumbres y el orden público. Debe abstenerse de usos ilícitos o malintencionados que puedan perjudicar a terceros o impedir el funcionamiento normal del sitio. El acceso al Sitio Web es gratuito salvo los costes de conexión a Internet. La empresa puede exigir el envío de formularios o autorización previa para determinados servicios. El usuario garantiza la veracidad y actualidad de los datos que facilite.

Conductas prohibidas (entre otras):

- Destruir o dañar datos, programas o documentos del Sitio Web.
- Obstaculizar el servicio mediante consumos masivos de recursos o acciones que generen errores.
- Introducir virus o secuencias lógicas que alteren sistemas propios o de terceros.
- Obtener contenidos a los que no tenga derecho o emplearlos con fines ilícitos.
- Difundir contenidos discriminatorios, violentos, pornográficos o contrarios a los derechos humanos.
- En caso de registrarse, usar información veraz y custodiar las claves de acceso.

El sitio actuará con diligencia para retirar o bloquear el contenido.
